AHL Man Systematic Strategies

Welcome to AHL Man Systematic Strategies Pages.

AHL Man Systematic Strategies runs on open source technology.

From Linux as the basis for our research cluster, to MongoDB for high speed data management, to Python and a range of analytic libraries for our research and trading systems all the way to AngularJS, D3 and related frameworks for browser-based visualisations.

Open Source Participation

We contribute back to the community by either releasing entire frameworks, such as MDF, or enhancing existing projects such as PyDev. We believe that by participating in the open source community we can build awareness of our team and technology, and attract talented individuals to join us; in addition, contributions from others using our technology may help us in the future.

Projects

Current projects we contribute to include:

MDFa data-flow programming framework in Python
PyDeva Python plugin for Eclipse
Mockextrastubs and matchers for the mock Python framework
numpya scientific package for Python
pandasa data analysis library for Python
